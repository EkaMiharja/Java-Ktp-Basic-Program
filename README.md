# KTP Output Program (Java)

## Description
This project is a simple Java console application that displays Indonesian ID card (KTP) information using static output.  
The program is **intended for beginner-level Java learners** to practice basic console output and understand program structure.

## Purpose
- Learn how to use `System.out.println`
- Understand basic Java class and `main` method structure
- Practice formatting text output in the console
- Get familiar with simple Java syntax

## Features
- Displays KTP information in a formatted layout
- Uses static (hardcoded) data
- Simple and easy-to-understand code
- Suitable for first-time Java learners

## Program Explanation

### Class: `ktpoutput`
This class contains the main program logic to display KTP data.

### Main Method
The `main` method:
- Prints KTP header information (province and city)
- Displays identity details such as:
  - NIK
  - Name
  - Birth information
  - Gender and blood type
  - Address
  - Religion
  - Marital status
  - Job
  - Nationality
  - Validity period
- Uses `System.out.println` for formatted console output

## How to Run
1. Clone or download this repository
2. Compile the Java file:
  javac ktpoutput.java
3. Run the program:
  java ktpoutput

## Notes
1. This program does not use user input
2. All data is static and hardcoded
3. Designed only for learning basic Java output, not real data processing
